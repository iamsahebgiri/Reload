Getting started

There are some ways to get started. First, see all download options available below, then choose the most suitable option for your need. Now you should add the main file of the Reload and the CSS Reset in the header of your project. Just that!

Install with npm

Milligram is also available to install using npm.

npm install @iamsahebgiri/reload

Install with yarn

Milligram is also available to install using yarn.

yarn add @iamsahebgiri/reload

Usage

First, use any method mentioned above to download Milligram. Then, just add these tags in the head. Milligram is also available via CDN.

Typography

CSS3 introduces a few new units, including the rem unit, which stands for "root em". The rem unit is relative to the font-size of the root element html . That means that we can define a single font size on the root element, and define all rem units to be a percentage of that. The typography has font-size defined in 1rem (16px) and line-height in 1.6 (24px). Milligram uses the font-family Inter provided by Google.

Button

The Button, an essential part of any user experience. Buttons come in three basic styles in Milligram: The button element has flat color by default, whereas .button-outline has a simple outline around, and .button-clear is entirely clear.
